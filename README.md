# Дипломный проект по теме Unit-тесты на курсе "Автоматизатор тестирования на Java"

### Технологии проекта:

Java 11, JUnit4, Maven, Jacoco, Mockito

### Как запускать:

`mvn clean test`

параметризацию, моки и стабы где необходимо.

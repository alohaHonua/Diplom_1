Описание работы
----------------

Данная часть диплома (Диплом 1) является набором unit-тестов для программы, которая помогает заказать бургер в Stellar Burgers.
Тестами покрыты классы: 
- Bun, 
- Burger, 
- Ingredient, 
- IngredientType. 
В тестах использованы моки, стабы и параметризация.

Используемые технологии
-----------------------

| Java 11 |
| Maven 3.9.0 |
| JUnit 4.13.2  |
| Mockito 4.11.0 |
| Jacoco 0.8.8 |

Запуск
------

Для запуска тестов необходимо воспользоваться командой "mvn clean test".

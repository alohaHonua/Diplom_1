# Описание
Проект предназначен для юнит-тестирования классов приложения Stellar Burgers.

Используемые технологии: 
- Java 11
- maven 3.9.0
- JUnit 4
- Jacoco
- Mockito 3.12.4

# Запуск
Команда для запуска - `mvn clean test`.
После запуска будут проверены тесты и сформируется отчет Jacoco.
